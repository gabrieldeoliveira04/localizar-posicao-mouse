# Mouse Position Locator Automation

Projeto desenvolvido em Python utilizando a biblioteca PyAutoGUI com o objetivo de localizar e identificar a posição do cursor do mouse na tela para utilização em automações desktop.

A aplicação foi criada como ferramenta auxiliar para projetos de automação, permitindo capturar coordenadas específicas da tela utilizadas posteriormente em scripts automatizados.

O sistema auxilia no desenvolvimento de automações baseadas em interação visual com elementos da interface, sendo útil para tarefas repetitivas, automações administrativas e fluxos automatizados.

## Objetivo do Projeto

O principal objetivo do projeto foi desenvolver uma ferramenta simples para identificar posições do mouse na tela e auxiliar na construção de automações desktop utilizando Python.

Além disso, o projeto também teve como foco praticar automação desktop, manipulação de coordenadas de tela e utilização da biblioteca PyAutoGUI.
